               6.2-C  Mathematical symbol input  (three complementary mechanisms)

                 Mechanism 1 — Live substitution (MathSymbolTypedHandler)
                 A TypedHandlerDelegate fires on every keypress. When the caret is
                 immediately after a `\`-prefixed identifier (e.g. `\forall`), typing
                 Space or Tab replaces the escape + trigger key with the Unicode symbol.
                 Example: user types `\forall<Space>` → editor inserts `∀ ` in-place.

                 The full escape table (MathSymbolTable.kt):
                   \forall  → ∀    \exists → ∃    \sum   → ∑    \prod  → ∏
                   \in      → ∈    \and    → ∧    \or    → ∨    \not   → ¬
                   \pi      → π    \inf    → ∞    \->    → →    \leq   → ≤
                   \geq     → ≥    \neq    → ≠    \div   → ÷    \xor   → ⊕
                   \lambda  → λ    \alpha  → α    \beta  → β    \gamma → γ
                   \delta   → δ    \epsilon→ ε    \theta → θ    \sigma → σ
                   \omega   → ω    \mu     → μ    \tau   → τ    \rho   → ρ

                 Mechanism 2 — Completion popup (MathSymbolCompletionContributor)
                 A custom CompletionContributor activates inside .nv files when the
                 prefix starts with `\`. It injects LookupElements for all matching
                 symbols; each shows the symbol glyph + escape name in the popup.
                 Selecting any entry inserts the Unicode code point and removes the
                 `\` prefix. Works naturally with Ctrl+Space.

                 Mechanism 3 — Insert Symbol action
                 "Nordvest: Insert Math Symbol" action (Alt+Backslash default shortcut)
                 opens a searchable symbol picker dialog. Shows all symbols in a grid
                 with name and codepoint; double-click or Enter inserts at caret.
